# Observer

Minimal Node.js application scaffold using TypeScript and Express.

## Scripts

- `npm run dev` starts the app and reloads when `.ts` files in `src/` change
- `npm run build` compiles TypeScript to `dist/`
- `npm run start` runs the compiled app
- `npm run typecheck` validates types without emitting files

## App

The app serves a static HTML page from `/` on port `3000` by default.
